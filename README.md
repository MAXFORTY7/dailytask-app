# Task Gamification App (Flutter)

Diese App entsteht als persönliches Projekt während meines Medieninformatik-Studiums.

🎯 **Ziele der App:**
- Tägliche und wöchentliche Tasks verwalten
- Prioritäten setzen
- Motivation durch **Streaks, Awards und Gamification-Elemente** steigern

👨‍💻 **Technischer Fokus:**
- Flutter
- State Management
- Lokale Datenspeicherung
- Sauberes UI/UX-Design

Dieses Projekt dient gleichzeitig:
- als **Portfolio-Projekt**
- als Vorbereitung für meine **Bachelorarbeit (Physio Motiva Move App)**
